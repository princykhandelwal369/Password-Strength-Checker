🔐 Password Strength Checker
A simple web application that checks the strength of a password based on multiple security rules. Built using HTML, CSS, and JavaScript.

🚀 Features
- Checks password length
- Validates uppercase letters
- Validates lowercase letters
- Checks numbers
- Checks special characters
- Gives a strength rating (Weak / Medium / Strong)
- Visual strength bar indicator

🛠️ Tech Stack
- HTML
- CSS
- JavaScript

📸 How it works
1. Enter a password in the input box
2. Click on "Check Strength"
3. The app evaluates the password
4. You get a strength score + visual feedback bar

💡 What I learned

- DOM manipulation in JavaScript
- Conditional logic and scoring systems
- Connecting frontend components (HTML + CSS + JS)
- Basic UI/UX feedback design
